# HR Employee Attrition Analysis Dashboard

## Project Overview
This project analyzes employee attrition patterns using HR data. 
The goal of the dashboard is to identify key factors that contribute to employee turnover and provide insights that can help organizations improve employee retention.

## Tools & Technologies Used
- Excel – Data cleaning and preprocessing
- MySQL – Data querying and extraction
- Power BI – Data visualization and dashboard creation

## Dashboard Features
The interactive dashboard provides insights into:

- Overall employee attrition rate
- Attrition distribution across departments
- Attrition by job role
- Age group distribution of employees
- Impact of business travel on attrition
- Relationship between salary bands and attrition

## Key Insights
- The Research & Development department shows the highest attrition.
- Employees aged 26–35 have the highest number of employees leaving.
- Employees who travel frequently have a higher attrition rate.
- Lower income bands show higher employee turnover.
